Welcome to Wi-Fi ToolBox Help

Main Page:

  • Connected WiFi: Displays the name of the currently connected WiFi network.
  • Connected WiFi Password: Provides the password for the currently connected WiFi network.
  • Saved WiFi Networks: Lists all previously saved WiFi networks for your reference.
  • QR Code for Connected WiFi: Tap to view the QR code for the currently connected WiFi network.
  • QR Code for Selected WiFi: Displays the QR code for a selected WiFi network from the saved list.
  • Password for Selected WiFi: Shows the password for a selected WiFi network from the saved list.

Utilities:

  • Wi-Fi Credentials: Click on the connected WiFi name to view its password and its QR Code.
  • Wi-Fi Credentials: Click on one WiFi from the List to view its password and its QR Code.
  • Copy to Clipboard: Right-click on the connected WiFi name to copy its password to the clipboard.

Menu:

  • Language Selection:

    • English: Switches the app's language to English. Your preference will be remembered for future app sessions.
    • Greek: Switches the app's language to Greek. Your preference will be remembered for future app sessions.

  • Generate and Save WiFi Credentials PNG:

    • Allows you to generate a PNG file with WiFi credentials, including the name, password, and QR code.
    • Choose a directory to save the PNG file.
    • The generated PNG will display the credentials of the connected WiFi network by default. If you select a WiFi network from the saved list, it will show the credentials for the selected WiFi instead.
